# クラスタ 4: データ前処理 & クリーニング自動化

[← index.md](index.md)

## 概要

LLMの常識推論能力を活用して、データクリーニング・欠損値処理・型変換・標準化などのデータ前処理を自動化する研究領域。従来のルールベース手法からプロンプト駆動・エージェント型ワークフローへの転換が進行中。テーブルデータの品質改善、非構造化データからの構造化抽出も含む。

## キーワード

`data cleaning`, `data wrangling`, `data preprocessing`, `data standardization`, `CleanAgent`, `AutoDCWorkflow`, `missing value imputation`, `schema parsing`, `anomaly detection`, `data quality`, `tabular data`
