- リソース数: 11

| # | 種別 | 年 | タイトル | 要約 / 本コンペへの関連 |

| 1 | paper | 2024 | [Learning With Generalised Card Representations for "Magic: The Gathering"](https://arxiv.org/abs/2407.05879) | 未知のカードにも汎化するカード表現を学習する研究。数値・名義・テキスト特徴、カード画像、第三者サービスのメタ情報を入力とし、Siamese ネットワークによる comparison training で表現を獲得。未見カードに対し人間のカード選択の55%を予測できることを示した。<br>**関連**: generalised card representation / card embedding の中核論文。本コンペは数千種のカードを扱い未知の組合せに汎化する必要があり、固定カード集合に依存しない埋め込み設計の指針として直接有用。 |
| 2 | paper | 2022 | [Towards Modern Card Games with Large-Scale Action Spaces Through Action Representation](https://arxiv.org/abs/2206.12700) | 約2,300万通りという巨大な離散行動空間を持つカードゲーム Axie Infinity を対象に、行動表現を学習するハイブリッドRLフレームワークを提案。全行動を評価する代わりに行動表現で定まる固定サイズ集合内のみを評価し、勝率とサンプル効率で最良を達成。<br>**関連**: 本コンペの巨大組合せ行動空間に直接対応する手法。合法手 index を扱う agent 設計において、parameterized/learned action representation により探索を縮約するアイデアの一次資料。 |
| 3 | paper | 2021 | [Evolving Evaluation Functions for Collectible Card Game AI](https://arxiv.org/abs/2105.01115) | CCG「Legends of Code and Magic」を題材に、評価関数の進化的学習を検討。線形重み型と2種のツリー型ゲノム表現を比較し、弱い/強い/進化中の対戦相手に対する fitness 評価戦略の影響を分析。他ドメインへの適用可能性も示唆。<br>**関連**: evolving evaluation functions / board evaluation function のキーワードに直結。本コンペでヒューリスティック盤面評価関数を進化的に最適化する際の表現選択・対戦相手選択の設計知見を提供。 |
| 4 | paper | 2024 | [A Taxonomy of Collectible Card Games from a Game-Playing AI Perspective](https://arxiv.org/abs/2410.06299) | CCG をルール・メカニクス・ゲームモードの観点から分類した taxonomy 論文（Vieira, Tavares, Chaimowicz）。ゲームAI研究の視点で各ゲームの特性を整理し、不完全情報・確率性・行動空間などの軸で位置づけを与える。<br>**関連**: ポケカを既存CCG研究の文脈に位置づけ、状態・行動表現や評価関数の設計判断を行う際の俯瞰図として有用。手法選定の前提整理に役立つサーベイ。 |
| 5 | paper | 2017 | [Deep Sets](http://papers.neurips.cc/paper/6931-deep-sets.pdf) | 集合を入力とする置換不変な関数を f(S)=ρ(Σ φ(s)) の形で表現する基盤論文。要素ごとの埋め込みをプーリングで集約することで、要素数可変かつ順序非依存の集合表現を学習可能にする。<br>**関連**: set-based encoding の理論的基礎。手札・ベンチ・トラッシュなど順序が意味を持たずサイズが変動するゾーンを置換不変に符号化する state encoding 設計に直接適用できる。 |
| 6 | paper | 2019 | [Set Transformer: A Framework for Attention-based Permutation-Invariant Neural Networks](https://arxiv.org/pdf/1810.00825) | Attention に基づく置換不変な集合処理アーキテクチャ。単純なプーリングと異なり集合内要素間のペアワイズ相互作用をモデル化でき、誘導点を用いて計算量を抑えつつ高表現力な集合エンコーダを実現。<br>**関連**: set-based encoding の高度版。盤面上のポケモン・エネルギー・手札カード間の相互作用（相性・シナジー）を捉える state/board 表現の実装候補として有用。 |
| 7 | paper | 2023 | [Two-Step Reinforcement Learning for Multistage Strategy Card Game](https://arxiv.org/html/2311.17305v1) | Legends of Code and Magic を対象に、ドラフト段階と対戦段階を分けた二段階RLを提案。MLP で選択済みカードを状態ベクトルに含める変種や、LSTM で逐次情報を蓄積する変種など、状態表現の違いを比較検討。<br>**関連**: multistage なカードゲームにおける state encoding の具体設計例。デッキ準備と対戦という段階構造を持つ本コンペで、状態ベクトル化と系列情報の扱いを設計する参考になる。 |
| 8 | competition | 2019 | [Introducing the Hearthstone-AI Competition](https://arxiv.org/pdf/1906.04238) | オンラインCCG Hearthstone を題材とした自律エージェント開発コンペ（2018-2020）の紹介。既知/未知デッキを用いる pre-made deck トラックと、事前にデッキ自作する self-made deck トラックを提供し、不完全情報CCGのAIベンチマークを整備。<br>**関連**: 本コンペと同種の不完全情報CCG先行コンペ。状態表現・行動マスキング・評価関数を含むエージェント設計の前例として、ルール構造やベンチマーク運用の参考になる。 |
| 9 | competition | 2017 | [Helping AI to Play Hearthstone: AAIA'17 Data Mining Challenge](https://arxiv.org/pdf/1708.00730) | Hearthstone の盤面状態のみから勝率を予測する state evaluation function を開発するデータマイニングコンペの報告。ミニオンHP・攻撃力・taunt 等の低次特徴を大きなベクトルに符号化する状態表現と予測手法を比較した。<br>**関連**: board evaluation function を学習で構築する直接的な先行事例。本コンペで状態→勝率の評価関数を作る際の特徴量設計（盤面・プレイヤー・手札属性のベクトル化）の具体的指針となる。 |
