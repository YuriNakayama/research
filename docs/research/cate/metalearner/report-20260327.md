# MetaLearnerによるCATE推定：最新論文調査（2024-2025年）

2024年以降に発表された主要論文5本を特定しました。

## 1. Meta-Learners for Partially-Identified Treatment Effects Across Multiple Environments
- **著者**: Schweisthal, Frauen, van der Schaar, Feuerriegel
- **発表**: **ICML 2024**
- **リンク**: [arXiv:2406.02464](https://arxiv.org/abs/2406.02464)

複数環境（異なる病院・国等）の観察データからCATEを推定する問題に取り組み、overlap・unconfoundedness仮定が成立しない状況に対応。環境をIVとして活用し、**モデル非依存型メタラーナー**で部分識別の境界を推定する手法を提案。メタラーナーの適用範囲を**部分識別**領域へ拡張した画期的研究。

## 2. Multi-Study R-Learner for Estimating HTE Across Studies
- **著者**: Shyr, Ren, Patil, Parmigiani
- **発表**: *Biostatistics* Vol.26, **2025年**
- **リンク**: [Oxford Academic](https://academic.oup.com/biostatistics/article/26/1/kxaf040/8383358)

R-learnerを複数研究設定に拡張した**Multi-Study R-learner**を提案。研究間の3種類の異質性（CATE、期待ポテンシャルアウトカム、処置割付メカニズム）を考慮し、データ適応型目的関数で研究間の情報借用を実現。精密医療での一般化可能性向上に貢献。

## 3. A Meta-Learner for Heterogeneous Effects in DiD
- **著者**: Lan, Chang, Dillon, Syrgkanis
- **発表**: arXiv, **2025年2月**
- **リンク**: [arXiv:2502.04699](https://arxiv.org/html/2502.04699v2)

DiD設定下でCATTを推定する**二重頑健メタラーナー**を開発。識別方程式を凸損失最小化問題に変換し、Neyman直交性による頑健性を確保。合成データで**MSE 50-75%改善**を達成。パネルデータの因果推論にメタラーナーを本格導入した先駆的研究。

## 4. RXlearner（Hölder連続性下のメタラーナー）
- **著者**: Zhao, Zhou
- **発表**: *Mathematics* (MDPI), **2025年5月**
- **リンク**: [MDPI](https://www.mdpi.com/2227-7390/13/11/1739)

X-learnerの重み付けメカニズムを改良した**RXlearner**を提案。リプシッツ連続性より弱い**ヘルダー連続性**仮定の下で非漸近的誤差上界を導出し、達成可能性も証明。理論的適用範囲を大幅に拡張。

## 5. Comparing Meta-Learners for HTE Estimation
- **発表**: *Mathematical and Computational Applications* (MDPI), **2025年12月**
- **リンク**: [MDPI](https://www.mdpi.com/2297-8747/30/6/139)

S/T/X/DR/R-learnerとCausal Forestの包括的シミュレーション比較。結論：

| メタラーナー | 最適な条件 |
|---|---|
| S-learner | CATEがほぼゼロの場合 |
| T-learner | 応答関数が処置群間で大きく異なる場合 |
| X-learner | 不均衡サンプル |
| R/DR-learner | 高次元設定 |

## 最新動向まとめ

**4つの主要方向性**が見られます：
1. **仮定の緩和** — 部分識別、ヘルダー連続性への対応
2. **適用領域の拡張** — 複数研究統合、DiD、生存時間データ
3. **理論的基盤の強化** — 非漸近誤差上界、Neyman直交性
4. **実務的示唆** — BART+S-learnerが平均最良、データ特性に応じた選択が重要
