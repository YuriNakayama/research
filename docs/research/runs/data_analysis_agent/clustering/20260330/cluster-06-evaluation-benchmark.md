# クラスタ 6: 評価・ベンチマーク

[← index.md](index.md)

## 概要

データ分析エージェントの能力を客観的に測定するためのベンチマーク・評価フレームワークに関する研究。DataSciBench、MMTU、ScienceAgentBenchなど専用ベンチマークが2024〜2025年に続々登場。従来の静的ベンチマークから、多次元的・文脈依存的・継続的な評価フレームワークへの進化が見られる。

## キーワード

`DataSciBench`, `MMTU`, `ScienceAgentBench`, `DataGovBench`, `benchmark`, `evaluation metrics`, `execution accuracy`, `task completion rate`, `automated evaluation`, `data science automation`

## 研究戦略

- **推奨検索クエリ**:
  - `"DataSciBench" LLM data science benchmark`
  - `"data analysis agent" evaluation benchmark 2024 2025`
  - `"ScienceAgentBench" LLM`
- **主要情報源**: NeurIPS Datasets & Benchmarks, ICLR, arXiv (cs.AI)
- **注目すべきベンチマーク**:
  - [DataSciBench (OpenReview)](https://openreview.net/forum?id=BltaWJZMeR) — データサイエンスLLMエージェントの包括的ベンチマーク
  - [DataGovBench (arXiv:2512.04416)](https://arxiv.org/html/2512.04416v2) — 実世界データガバナンスワークフローのベンチマーク
  - [ScienceAgentBench](https://www.emergentmind.com/topics/scienceagentbench) — 科学的発見タスクのベンチマーク
- **推奨読み進め順**: サーベイ論文 [arXiv:2506.08800](https://arxiv.org/html/2506.08800v2) を起点に推奨
- **チュートリアル**: [KDD 2025 Tutorial: Evaluation & Benchmarking of LLM Agents](https://sap-samples.github.io/llm-agents-eval-tutorial/)
