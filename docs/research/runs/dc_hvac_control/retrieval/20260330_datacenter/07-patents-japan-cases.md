# クラスタ7: 特許動向・日本の実証事例

## 概要

データセンター冷却技術の特許動向は、空冷から液冷への劇的なパラダイムシフトを示している。2000-2023年の特許分析によれば、空冷特許は全体の69%から9%に減少し、液冷特許は24%から84%に急増した。日本では、NTTグループ（ファシリティーズ、データ、コミュニケーションズ）が液冷・AI空調制御の両分野で先行し、ダイキン工業はDDCS社買収により液冷分野に参入した。IIJはPUE 1.2を目標としたAI空調制御DCを運用中であり、高砂熱学工業のDC省エネ設計はASHRAE Technology Awards 2025で世界2位を受賞した。2029年の省エネ義務化が日本のDC業界全体の技術採用を加速させている。
