# 不正行動マスキング: 巨大行動空間における合法手のみへの確率割当

## 概要

Huang と Ontañón による「A Closer Look at Invalid Action Masking in Policy Gradient Algorithms」(arXiv:2006.14171, FLAIRS-35, 2022) は、複雑なルールを持つゲームで方策が出力する行動の多くが非合法になる問題に対し、**不正行動マスキング(invalid action masking)** が(1)正当な方策勾配更新に対応すること、(2)非合法行動空間が増えるほど重要になることを理論・実験で示した。手法は、非合法行動の logits を softmax 前に巨大な負値(例 -10^8)で置換し、合法手のみから確率的にサンプルする。PTCG の `agent(obs_dict)->list[int]` が**合法手 index のみ**を返す構造、および `obs_dict.select` で合法手集合が局面ごとに変わる構造に正面から対応する基礎技術である。

## 手法の要点

- **マスキング機構**: ネットの出力 logits のうち、非合法行動に対応する要素を **softmax 適用前に -∞(実装上 -1e8 等)に置換** する。結果、非合法行動のサンプリング確率がほぼ 0 になる。
- **理論的正当化**: マスク関数 inv_s は合法行動には恒等写像、非合法行動には定数関数を適用する。両者とも微分可能なので、マスク後の方策 π' はパラメータについて微分可能であり、**方策勾配定理の前提を満たす**。すなわちマスキングは「**状態依存の微分可能関数**」として作用し(通常の sigmoid/softmax と異なり、状態ごとに合法手集合が変わる)、正当な方策勾配を与える。
- **勾配の振る舞い**: 行動 a_2 がマスクされると **その勾配は 0 になり**、合法行動の勾配は非ゼロのまま残る。この選択的勾配修正が手法の有効性の本質。
- **対照手法**: 「不正行動ペナルティ(非合法行動に負報酬)」は、報酬経由でしか学習が進まず、合法手割合が小さくなるほど効率が悪化しスケールしない。

## 主要な結果や知見

- **microRTS** での資源採取タスク(4×4〜24×24 マップ)で評価。
- **スケーリング**: マスキングは収束時間 t_solve がマップサイズに依らず **約 12% で一定**。一方ペナルティ法はマップが大きく(合法手割合が小さく)なるほど劣化(合法手確率 4×4 で 0.125 → 24×24 で 0.0034)。
- **初期報酬到達速度**: 10×10 マップでペナルティ法は最初の報酬に全ステップの 3.43% を要したが、マスキングは **0.06%** で到達。
- **頑健性**: マスキングで学習したエージェントは、評価時にマスクを外しても許容できる性能を示した。
- 結論: 巨大かつ可変な行動空間では、不正行動マスキングがペナルティ法より圧倒的に効率的かつスケーラブルであり、理論的にも方策勾配を歪めない。

## 本コンペ(PTCG AI Battle)への応用

- **合法手 index への直接対応**: PTCG では `obs_dict.select` が合法手 index 集合を与え、エージェントは合法手のみを返す。方策ネットの全行動 logits に対し `select` に含まれない index を -∞ でマスクすれば、**常に合法手のみをサンプル/argmax** でき、非合法手を返してエラーになる事故を構造的に防げる。
- **局面ごとに変わる合法手集合**: PTCG は局面(ドロー・進化・エネルギー付与・技選択・デッキ選択)で合法手集合が大きく変動する。本論文の「状態依存の微分可能マスク」はまさにこの設定に合致し、固定長出力のネットで可変合法手を扱える。
- **巨大行動空間での学習効率**: 約2000枚プール由来の巨大行動空間では、非合法手が大多数を占める局面が頻出する。マスキングは非合法手割合が高いほど効果が大きいという知見から、PTCG の RL 学習を大幅に高速・安定化できる。
- **デッキ選択フェーズ(current=None)**: デッキ構築でも選べるカードが制約される。マスキングで合法な選択肢のみに確率を割り当てれば、60枚デッキの逐次構築を方策勾配で安定学習できる。
- **前述の行動表現手法との併用**: Wolpertinger/Axie 流の近傍 k 候補抽出後に、その候補内でマスキング+softmax を適用すれば、巨大行動空間の絞り込みと合法性保証を両立できる。
- **Gaussian ラダーでの信頼性**: 自動対戦ラダーで非合法手によるクラッシュや時間浪費を避けることは安定勝率に直結し、マスキングはその基礎保証を与える。
