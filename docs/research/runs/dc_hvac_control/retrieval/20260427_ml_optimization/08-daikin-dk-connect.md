# 08. Daikin "Remote Automatic Energy-Saving Control" / DK-CONNECT

## 概要情報

| 項目 | 内容 |
|------|------|
| サービス名 | Remote Automatic Energy-Saving Control（Air Net Service System / DK-CONNECT 統合） |
| 提供企業 | ダイキン工業株式会社 |
| 商用展開時期 | 2024年秋（Air Net）/ 2025年春（DK-CONNECT） |
| 公式リンク | https://www.daikin.com/about/corporate/tic/topics/feature/2025/20250514 |

## 位置付け

学術論文ではなく **商用 AI 制御サービスの実装事例**。学術界が提唱してきた ML ベース HVAC 最適化が、実運用ビジネスとして展開された代表例。論文との数値ギャップを理解する上で重要なベンチマーク。

## 技術アプローチ

- **熱負荷予測**: センサデータ + 気象情報から学習する自動予測モデル
- **建物特性同定**: 実際の冷却能力データを ML で解析し、建物パラメータを自動推定
- **予測制御 (Feed-Forward)**: 先行的最適化によるプロアクティブ制御
- **モデル更新**: 性能劣化を防ぐ定期的再学習

## 実証成果

| 案件 | 期間 | 省エネ率 |
|------|------|---------|
| ヤマハ施設 | 2023年4月〜2024年3月 | **21%** |
| BMW Thailand | 2024年1月〜6月 | **16%** |

両案件とも「快適性に顕著な差なし」と報告。

## 展開スコープ

- **主対象**: 商業向けマルチスプリットエアコン
- **拡張計画**: ルームエアコンから大型ターボ冷凍機（centrifugal chiller）まで
- **対象セクター**: オフィス、病院、商業施設

## 主要機能

- 自動・遠隔制御最適化
- **運用工数を約 90% 削減**
- 建物仕様データの手動入力が不要

## 意義

1. **学術と実務の数値ギャップ**: 学術論文は 26-45% の省エネを謳うが、実商用環境では 16-21% に収束する傾向。これはレビュー論文 [09](./09-field-demo-review.md) の指摘とも整合。
2. **商用化の到達点**: AI/ML 制御は研究段階から **マネージドサービス化** へと移行している。
3. **保守コスト削減**: 単純な省エネ率だけでなく、**運用人件費の削減**も訴求点となる。

## 限界・留意点

- 詳細なアルゴリズムは非公開（営業秘密）。
- 上記省エネ率は単一サイト報告で、業界平均ではない。
- ターボ冷凍機への展開は予定段階で、実証はこれから。
