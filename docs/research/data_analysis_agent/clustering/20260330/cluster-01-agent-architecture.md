# クラスタ 1: エージェントアーキテクチャ & 推論基盤

[← index.md](index.md)

## 概要

LLMベースのデータ分析エージェントを支えるコアアーキテクチャに関する研究領域。Plan-then-Execute（計画→実行）型、ReAct（推論→行動）型のフレームワーク設計、リフレクション（自己評価・修正）メカニズム、ツール使用（Function Calling）、RAG（検索拡張生成）によるドメイン知識統合などが主要テーマ。エージェントが複雑なデータ分析タスクを自律的に分解・実行する能力の基盤となる。

## キーワード

`Plan-then-Execute`, `ReAct`, `Chain-of-Thought (CoT)`, `reflection`, `self-correction`, `tool use`, `function calling`, `Agentic RAG`, `knowledge integration`, `reasoning`, `LLM agent architecture`, `prompt engineering`, `error handling`
