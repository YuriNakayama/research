# 2. Transfer Learning for Individual Treatment Effect Estimation (CITA)

- **URL / arXiv**: https://arxiv.org/abs/2210.00380
- **著者 / 発表年 / venue**: Ahmed Aloui, Juncheng Dong, Cat P. Le, Vahid Tarokh / 2022（2023改訂）/ arXiv (cs.LG)

## 概要
個別処置効果（ITE）推定において、あるタスクで得た因果知識を別タスクへ転移する枠組みを提案する。ITE は反実仮想（counterfactual）が観測できないため転移が理論的に難しいが、著者らは counterfactual loss と ITE 誤差に対する汎化バウンドを示し転移の実現可能性を裏付ける。中心的貢献は因果推論タスク間の類似度を測る新指標 CITA（Causal Inference Task Affinity）で、これを用いて最も近いソースタスクを選び転移する。ITE 推定に必要なデータ量を最大 95% 削減できたと報告する。

## 手法・キーアイデア
CITA はターゲットタスクに対する各ソースタスクの「因果的近さ（task affinity）」を定量化する指標である。タスク親和性の考え方（Le らの Task Affinity 系）を因果推論設定へ拡張し、反実仮想情報の欠損を汎化バウンドで扱う。運用は明快で、(1) 複数のソースタスク候補に対し CITA を計算、(2) 最も親和性の高いソースを選択、(3) そのソースで学習した表現／モデルをターゲットへ転移して ITE を推定する。ソース選択を明示的な指標に基づかせる点が核心。

## ユーザー課題への適用
本手法は「複数の旧施策のうち、どれを新施策の転移元にすべきか」という転移元選択問題に直接答える。過去に多数のマーケティング施策の CATE/ITE モデルがある場合、新施策（ターゲット、データ僅少）に対し各旧施策の CITA を算出し、最も因果構造が近い施策を転移元に選べる。恣意的にソースを選ぶと起きる negative transfer を、親和性最大のソースを選ぶことで暗黙的に回避する設計であり、複数施策資産の再利用戦略に理論的根拠を与える。データ量 95% 削減という結果は、新施策で学習用の実験配信を大幅に減らせる可能性を示す。

## 長所と限界
長所: 転移元選択を定量指標で自動化でき、少データ新施策での立ち上げに有効。汎化バウンドという理論的裏付けがある。限界: 特徴空間はソース・ターゲットで共通である前提が強く、特徴が異なる場合は 2210.06183 との併用が要る。CITA 計算コストと、選んだ 1 ソースのみ利用する（複数ソース融合をしない）点が制約。
