## 主要な知見

- **O1-previewが法的推論で最高性能**: 総合人間評価3.96で首位。特に英語事例で4.08と突出
- **自動指標と人間評価の乖離**: ROUGE最高のPhi-3.5-mini（0.41）は人間評価7位（2.62）。法的品質はn-gram一致度では測定不可
- **法律特化モデルの限界**: LawGPT_zh（2.00）やLawyer-LLaMA（2.58）は汎用モデルに劣る結果。ドメイン特化ファインチューニングだけでは不十分
- **コストパフォーマンス**: Qwen2-7B（オープンソース・7Bパラメータ）が人間評価3.85で2位。大規模クローズドソースモデルに匹敵
