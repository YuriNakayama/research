# クラスタ 3: マルチエージェント協調 & ワークフロー自動化

[← index.md](index.md)

## 概要

複数の専門化されたエージェント（データ収集、EDA、モデル構築、レポート生成など）が協調してエンドツーエンドのデータ分析パイプラインを構成するアプローチ。AutoGen、CrewAI、LangGraphなどのフレームワークが実装基盤として普及。ワークフローの自動オーケストレーション、エージェント間通信プロトコル、反復的改善ループが主要課題。
