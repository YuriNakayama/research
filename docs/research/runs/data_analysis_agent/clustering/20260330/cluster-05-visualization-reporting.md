# クラスタ 5: 可視化 & レポート自動生成

[← index.md](index.md)

## 概要

データ分析結果の可視化（チャート・グラフ・ダッシュボード）およびテキストレポートの自動生成に焦点を当てた研究領域。Microsoft LIDAのような包括的フレームワーク、NL2Vis（自然言語→可視化）、マルチモーダルフィードバックによる可視化品質改善などが含まれる。Chart Understanding（チャート読み取り）の逆方向の研究も関連する。

## キーワード

`NL2Vis`, `LIDA`, `data visualization`, `chart generation`, `automated reporting`, `dashboard generation`, `PlotGen`, `ChartLlama`, `multimodal feedback`, `infographic generation`
